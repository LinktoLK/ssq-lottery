## ✨ 功能亮点

### 🎲 号码选择
- 红球区（1-33 选 6 个）+ 蓝球区（1-16 选 1 个）
- 点击球号即可选择，支持随机一注和清空重选
- 漂亮的渐变球体动画效果

### 🔮 AI 好运指数预测（搞笑版）
基于多维度算法计算你的号码运势：
- **和值分析** — 号码总和是否适中
- **奇偶平衡** — 阴阳调和程度
- **连号加成** — 缘分相连指数
- **大小比例** — 分布合理性
- **特殊号码玄学** — 8=发、6=顺、18=要发...
- **躲坑扣分** — 需要避开的"凶数"

**6 档运气等级**：
| 等级 | 分数 | 特点 |
|------|------|------|
| 🌟 天选之子 | 95-100 | 连路边的流浪猫看了都想蹭你两下 |
| 💫 运气爆棚 | 85-94 | 走在路上被钱砸中的概率提升300% |
| 🍀 小确幸 | 70-84 | 像食堂阿姨手抖时打的饭量 |
| 😐 一般般 | 50-69 | 像极了单身多年的你 |
| 😅 需要努力 | 30-49 | AI神算子叹了口气... |
| 💀 渣渣 | 0-29 | 这组号码的气场比你的前任还凉 |

### 📜 历史中奖穿越查询
- 内置 50+ 期真实风格的历史开奖数据
- 自动匹配你的号码与历史记录的相似度
- 展示匹配的红/蓝球数量及对应奖项等级

### 📤 分享功能
- 一键生成分享文本（包含号码、分数、等级）
- 复制到剪贴板即可发给好友
